About This Version
Grape Statistics 0.2 is currently in an alpha development stage. Being so, you will encounter incomplete or buggy features. There will be no offical upgrades for development versions.

Requirements
Development versions are poorly tested on a variety of software. We recommend using anything higher than PHP 4.3 and MySQL 4.x.

How To Install
Edit the values in quotation marks in the /includes/connect.php file. They should be edited to connect with your MySQL database. Then run /includes/setup.php to set up the databases.

How To Track Statistics
Place the following in all pages you want to track statistics:
<script src="insert-location-to-grape/?js" type="text/javascript">
</script>
Replace "$location" with the location of your grape install.

How It Works
There are two main parts to this program: the data collector, and the parser. As indicated by the names, the data collector saves user information to a database, and the parser displays the statistics as a whole.

Troubles with Timezone?
Check out this list and edit your /includes/config.php file.
http://us3.php.net/manual/en/timezones.others.php

How To Manually Install Extensions
Most extensions will probably not need write permissions except database access. However, for an extension to be used in Grape, it must be added to the /extensions/extensions.php file, like so:
<?php
$enabled_ext[] = "GrapePages";
$enabled_ext[] = "GrapeReferrers";
?>
